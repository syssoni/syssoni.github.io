Music Recommender System

This is a content-based recommendation system (UI using streamlit) that lets users pick a song from a catalog and get five suggestions for similar tracks. The recommendations are based on matching genres and artist names, using precomputed cosine similarity scores and a saved dataset with song metadata.

  • Open Project

Spam Classifier

This is a email/sms spam classifier web application made using Streamlit and Natural language processing (including tokenization, stop word removal and lemmatization).It uses a TF-IDF vectorizer along with a machine learning model (loaded with Pickle) to classify messages entered by users as “Spam” or “Not Spam” instantly.

  • Open project

Virtual Meeting Analyser

This meeting video analyzer uses NLP and Hugging Face models to break down meeting transcripts. It figures out who’s speaking, checks the overall vibe with sentiment analysis, and picks out important keywords. Then, it puts together summaries and helps decide if the workplace vibe is healthy or toxic.

  • Open project

Helmet Detection

A real time helmet detection system made using yolov11 and yolov12 object detection models. This accurately identifies whether a the driver or the passenger is wearing helmet or not.
